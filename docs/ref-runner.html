Runner & Structure

System

Operating Systems

Define which OS the runner should use with runs-on.

runs-on: ubuntu-latest # Most Common
runs-on: windows-latest
runs-on: macos-latest
Basic

Full Basic Template

Every workflow file should follow this structural pattern.

name: My Workflow

on: [push]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - run: echo "Hello World"
