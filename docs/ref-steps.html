Steps & Commands

Run

Running Commands

Use the run key to execute shell commands.

# Single command
- run: npm install

# Multiple commands (use the pipe '|')
- run: |
    npm install
    npm run build
    npm test
Env

Steps with Env Variables

You can pass environment variables to individual steps.

- run: npm run build
  env:
    NODE_ENV: production
Context

Built-in Variables

GitHub provides useful information about the run.

  • ${{ github.repository }} - owner/repo
  • ${{ github.ref }} - branch name
  • ${{ github.actor }} - who started it
