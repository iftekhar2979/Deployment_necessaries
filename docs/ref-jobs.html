Jobs & Dependencies

Needs

Job Dependencies

Use needs to ensure one job waits for another to finish successfully.

jobs:
  build:
    runs-on: ubuntu-latest
    ...

  deploy:
    needs: build
    runs-on: ubuntu-latest
Matrix

Build Matrix

Run your jobs across multiple versions of languages or operating systems at once.

strategy:
  matrix:
    os: [ubuntu-latest, windows-latest]
    node-version: [18, 20, 22]
Flow

Conditional Execution

Control when jobs or steps run using if statements.

if: github.ref == 'refs/heads/main'
if: failure() # Only if previous job failed
