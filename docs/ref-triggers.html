Event Triggers

Triggers (defined by the on: key) determine when your workflow starts.

Push

When code is pushed

Runs whenever code is pushed to specific branches.

on:
  push:
    branches: [main, develop]
PR

When a Pull Request is created

Runs when a PR is opened or updated against target branches.

on:
  pull_request:
    branches: [main]
Schedule

Scheduled Tasks (Cron)

Runs at specific times using cron syntax.

on:
  schedule:
    - cron: '0 9 * * *' # 9 AM every day
Manual

Manual Trigger

Allows you to run the workflow manually from the GitHub UI.

on: workflow_dispatch
