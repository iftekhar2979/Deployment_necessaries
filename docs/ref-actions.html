Popular Actions (Recipes)

Essential

Checkout Code

Always used to download your repo's code onto the runner.

- uses: actions/checkout@v4
Environment

Setup Node.js

Installs a specific version of Node.js and npm.

- uses: actions/setup-node@v4
  with:
    node-version: 20
Deployment

Connect to Server (SSH)

Remotely connect to your server to run deployment commands.

- uses: appleboy/ssh-action@v1.0.3
  with:
    host: ${{ secrets.HOST }}
    username: ${{ secrets.USER }}
